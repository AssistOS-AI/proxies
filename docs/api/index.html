Soul Gateway API Reference
Reference

API Reference

Complete endpoint documentation for Soul Gateway. All management endpoints are served under /api/v1/ and require dashboard authentication. Proxy endpoints use Bearer token authentication with an API key.

Proxy Endpoints

Client-facing endpoints for LLM inference. Authenticated via Authorization: Bearer <api-key>.

POST

Chat Completions

/v1/chat/completions — OpenAI-compatible chat completion with full pipeline processing, middleware, tiering, and cost tracking.

 POST

Anthropic Messages

/v1/messages — Passthrough to Anthropic Messages API format providers.

 POST

OpenAI Responses

/v1/responses — Passthrough to OpenAI Responses API format providers.

 GET

List Models

/v1/models — OpenAI-compatible model listing. Returns enabled models and tiers.

 GET

List Tiers

/v1/tiers — Returns enabled tiers with their member models and fallback configuration.

 GET

Health Check

/health — Returns gateway health status and uptime. No authentication required.

Key Management

Create, update, and revoke API keys. Each key has per-key rate limits and daily budgets.

CRUD

API Keys

List, create, update, revoke API keys. Reset daily budget counters. Keys are created with plaintext returned once.

Model Management

Configure model routing, pricing, tags, and provider mappings.

CRUD

Models

Create, update, toggle, and delete model configurations. List available tags and discover models from providers.

Provider Management

Manage upstream LLM providers, test connectivity, sync models, and configure OAuth authentication.

CRUD + OAuth

Providers

Full provider lifecycle: create, update, delete, test, sync, discover models. OAuth device flow and PKCE for managed-auth providers.

Tier Management

Group models into named tiers with priority ordering and fallback chains.

CRUD

Tiers

Create, update, toggle, and delete tiers. Assign model references and fallback models.

Middleware Management

Configure the request/response middleware pipeline per model or per tier.

Configuration

Middlewares

List, update, and rescan middlewares. Assign middlewares to models or tiers with custom settings and ordering.

Observability

Logs, metrics, agents, sessions, real-time streaming, and data export.

Logs

Logs & Streaming

Query call logs with filters. Real-time log streaming via WebSocket and SSE. Agent and session tracking with hierarchy views.

 Metrics

Metrics & Export

Cost breakdowns, usage stats, error rates, token trends, activity by API key. System metrics and bulk CSV/JSON export.
