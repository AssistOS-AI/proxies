Soul Gateway
OpenAI-Compatible LLM Proxy
v1.0

Route any LLM through a single, intelligent gateway.

Soul Gateway is an OpenAI-compatible proxy that connects to 15+ LLM providers. It handles authentication, rate limiting, daily budgets, content filtering, middleware pipelines, model tiering with automatic fallback, and real-time cost tracking — all through a single /v1/chat/completions endpoint.

Read the architecture API reference

Quick Start

Send your first request in three steps.

  1. Start the gateway with Ploinky or standalone: 
    ploinky start soul-gateway
  2. Create an API key via the dashboard at http://localhost:8042, or via the management API: 
    curl -X POST http://localhost:8042/api/v1/keys \
  -H "Content-Type: application/json" \
  -d '{"label": "my-key"}'
  3. Send a chat completion request: 
    curl http://localhost:8042/v1/chat/completions \
  -H "Authorization: Bearer sk-your-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "fast",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'

Core Capabilities

Routing

Multi-Provider Dispatch

OpenAI, Anthropic, Google, Copilot, Kiro, Groq, DeepSeek, and more. One endpoint, many backends. Automatic format conversion between provider APIs.

Tiers

Model Tiering & Fallback

Group models into named tiers with priority ordering. If a model hits rate limits or quota, the gateway automatically falls back to the next model in the tier chain.

Auth

OAuth Credential Pooling

Native OAuth device flow and PKCE for Copilot, Kiro, Codex, Gemini, and Anthropic. Multi-account rotation with automatic token refresh and quota tracking.

Pipeline

Middleware Framework

12 built-in middleware plugins: rate limiting, budget enforcement, caching, loop detection, content filtering, prompt injection, and more. Write your own as simple .mjs files.

Budgets

Cost Control

Per-key daily and monthly budgets. Token-based and per-request pricing. Real-time cost tracking by model, key, and agent. Budget enforcement blocks requests before they incur cost.

Observability

Real-Time Streaming

Live log streaming via WebSocket and SSE. Cost dashboards, error rates, token usage analytics. Full audit trail with request/response content, latency, and retry details.

Proxy Endpoints

Three OpenAI-compatible endpoints for different provider API formats.

EndpointFormatDescription
POST /v1/chat/completions OpenAI Chat Main pipeline with full middleware, tiering, and cost tracking
POST /v1/messages Anthropic Messages Passthrough to Anthropic-format providers
POST /v1/responses OpenAI Responses Passthrough to OpenAI Responses API providers
